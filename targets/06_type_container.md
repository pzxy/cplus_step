# array
![array](./images/array.svg)
[demo](./06_type_container_array.cpp)
array是c++ 11新增的序列容器，和其他容器的区别是，array的大小是固定的，无法动态扩展或者收缩。只允许访问或者替换存储的元素。

# vector
布局和array一样，但是大小可以动态变化。
